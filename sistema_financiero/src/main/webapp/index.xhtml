Por dónde empezamos

Te mostramos las opciones principales del sistema!

Catalogo de Cuentas

Esta opcion muestra las principales cuentas existentes en la base de datos,a la cual puede agregar mas y si desea.

Contabilidad

Menu donde pobra registrar los movimientos diarios que se realizan en la empresa.

Reportes

Generar reporte de estados financieros relizados duerante un perio, Balance Gneral, Estado de Resultados.

Empresas

Menú para agregar o editar las empresas a las que se les efectua la contabilidad.

Cargar Archivo

Herramienta para cargar archivos csv que contengan un nuevo catalogo de cuetas o una serie de transacciones de las diferentes cuentas.
